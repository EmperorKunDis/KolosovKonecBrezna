1. Prehled projektu

Cílem projektu je vytvořit webový nástroj pro porovnání dodavatelů energií. Nástroj umožní poradcům načítat ceníky dodavatelů, analyzovat klientské faktury pomocí AI/OCR a automaticky porovnat nabídky všech dodavatelů s výpočtem potenciální úspory.

Nástroj bude provozován jako interní webová aplikace se sdíleným přístupem pro více uživatelů (poradců). Primární workflow: upload faktury/ceníku → AI extrakce dat → automatické porovnání → export výsledku pro klienta.

Postup zpracování

Krok Popis Technologie
1. Upload Uživatel nahraje PDF ceník nebo fakturu přes webové rozhraní Angular + DRF FileUpload
2. Předzpracování Kontrola formátu, velikosti, konverze stránek na obrázky pdf2image, Pillow
3. AI extrakce Odeslání na Gemini Vision API s promptem pro extrakci strukturovaných dat Gemini Vision API
4. Fallback Pokud Gemini selže, použije se Tesseract OCR + NLP parsování Tesseract, regex
5. Validace Kontrola úplnosti dat, formátů, rozsahů hodnot Django validators
6. Ruční korekce Poradce zkontroluje a případně opraví extrahovaná data Angular formulář

Extrahovaná data z ceníku

Název dodavatele, název tarifu, sazba za VT (Kč/kWh), sazba za NT (Kč/kWh), měsíční fixní poplatek, stálý plat za příkon, daně a regulované složky, platnost ceníku od/do.

Extrahovaná data z faktury

Dodavatel, odběrné místo, spotřeba VT (kWh), spotřeba NT (kWh), aktuální cena celkem, fakturační období, distribuční sazba.

6. Moduly MVP

1

Správa ceníků

  • Upload PDF ceníků dodavatelů
  • AI/OCR automatická extrakce dat
  • Formulář pro ruční korekci
  • Přehled a filtrování ceníků v DB
  • Správa dodavatelů (CRUD)
  • Indikace platnosti ceníku
2

Analýza faktury

  • Upload faktury (PDF, obrázek)
  • AI extrakce spotřeby a ceny
  • Validace extrahovaných dat
  • Ruční korekce poradcem
  • Historie nahraných faktur
  • Podpora VT/NT sazeb
3

Porovnávací engine

  • Výpočet ročních nákladů u všech dodavatelů
  • Zohlednění VT/NT spotřeby
  • Fixní poplatky a stálý plat
  • Řazení od nejlevnějšího
  • Zvýraznění nejlepší nabídky
  • Výpočet absolutní i procentuální úspory
4

PDF export

  • Generování přehledu pro klienta
  • Profesionální vzhled s logem
  • Tabulka porovnání dodavatelů
  • Zvýraznění úspory
  • Datum a podpis poradce
5

Uživatelská správa

  • Registrace a přihlášení (email + heslo)
  • Role: administrátor a poradce
  • Admin: správa uživatelů, ceníků, nastavení
  • Poradce: upload faktur, porovnání, export
  • JWT autentizace
  • Reset hesla

7. Bezpečnost

Bezpečnostní opatření

  • HTTPS — veškerá komunikace šifrována pomocí TLS 1.3
  • Autentizace — JWT tokeny s expirací, refresh tokeny
  • RBAC — role-based access control (admin / poradce)
  • CSRF ochrana — Django CSRF middleware
  • Rate limiting — omezení počtu API požadavků
  • Šifrování — citlivé údaje z faktur šifrovány v DB
  • File validation — kontrola typu a velikosti nahrávaných souborů
  • SQL injection — Django ORM jako ochrana
  • CORS — omezení na povolené domény
  • Audit log — záznam přístupů a operací

8. Omezení MVP

Co NENÍ součástí MVP

  • Mobilní aplikace (iOS/Android) — pouze responsivní web
  • Automatický import ceníků z webů dodavatelů (scraping)
  • Integrace s bankovními systémy (ČSOB, FIO apod.)
  • Historické trendy a vývoj cen v čase
  • Notifikace o změně ceníků
  • Vícenásobné odběrné místo v jedné analýze
  • Porovnání plynových a elektřinových dodavatelů v jednom reportu
  • Multi-tenant oddělení dat mezi různými poradenskými firmami

Výše uvedené funkce mohou být realizovány v rámci Fáze 2 rozšíření systému, po úspěšném nasazení a validaci MVP.
