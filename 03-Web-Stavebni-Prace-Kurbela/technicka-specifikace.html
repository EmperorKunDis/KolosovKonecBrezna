1 Přehled projektu

Předmětem projektu je vytvoření jednoduchého, moderního one-page webu pro stavební firmu Mykhaylo Kurbela (OSVČ). Web bude sloužit jako hlavní online prezentace firmy s cílem získávat nové klienty a prezentovat dosavadní realizace.

Responzivní design Ano
Cílová rychlost načtení < 3 sekundy
Termín dodání 2 týdny od zaplacení zálohy a dodání podkladů

2 Struktura webu - sekce

Web bude obsahovat následující sekce řazené vertikálně na jedné stránce se smooth scrollem:

2.1 Hero sekce

  • Velký vizuální prvek (fotografie realizace / profesionální obrázek stavby)
  • Hlavní nadpis s klíčovým sdělením firmy
  • CTA tlačítko "Nezávazná poptávka" s odkazem na kontaktní formulář
  • Případně doplňkový text (slogan, krátký popis)

2.2 O nás

  • Krátký text představující firmu, zkušenosti a hodnoty
  • Případně fotografie týmu nebo majitele
  • Klíčové USP (unikátní prodejní argumenty)

2.3 Služby

  • Přehledný výpis stavebních prací formou karet s ikonami
  • Minimální rozsah: rekonstrukce, novostavby, zateplení, podlahy, elektroinstalace, malířské práce
  • Každá služba: ikona, název, krátký popis

2.4 Reference / Galerie

  • Grid s fotografiemi realizací
  • Každá reference: fotografie, název projektu, krátký popis
  • Lightbox pro zvětšení fotografií (vanilla JS)

2.5 Kontakt

  • Kontaktní formulář (jméno, e-mail, telefon, zpráva)
  • Odesílání dat na e-mail objednatele
  • Kontaktní údaje firmy (adresa, telefon, e-mail)
  • Napojení na Google Maps s vyznačením adresy firmy

3 Technologie

Moderní stylování, flexbox, grid, animace

Interaktivita bez závislostí na knihovnách
Responsive Design
Mobile-first přístup, media queries

Bez externích závislostí

Web nebude záviset na žádných externích knihovnách ani frameworcích (bez jQuery, Bootstrap, React apod.). To zajišťuje maximální rychlost načítání a minimální údržbu do budoucna.

Doporučená varianta: Cloudflare Pages + vlastní doména. Nejnižší provozní náklady při vynikajícím výkonu a zabezpečení.

5 SEO a viditelnost

Základní SEO optimalizace

  • Meta tagy - title, description, keywords pro každou sekci
  • Structured Data - JSON-LD schema LocalBusiness (název firmy, adresa, telefon, otevírací doba, služby)
  • Sitemap.xml - pro indexaci vyhledávači
  • robots.txt - správné nastavení přístupu crawlerů
  • Open Graph tagy - pro správné zobrazení při sdílení na sociálních sítích
  • Sémantické HTML - správné použití heading hierarchie (h1-h3)

Výkon

  • Optimalizované obrázky (WebP formát, lazy loading)
  • Minifikovaný CSS a JS
  • Cílové skóre Google PageSpeed Insights: 90+
  • Celková velikost stránky: pod 1 MB

Způsob odesílání

Formulář bude odesílat data na e-mail objednatele. Technické řešení dle zvoleného hostingu:

  • Cloudflare Pages: Cloudflare Workers (serverless funkce) pro zpracování a odeslání e-mailu
  • Shared hosting: PHP skript pro odeslání e-mailu přes mail()

Formulář bude obsahovat validaci na straně klienta (HTML5 + JS) i na straně serveru.

9 Rozsah - co NENÍ součástí dodávky

  • Redakční systém (CMS) - web je statický, úpravy provádí zhotovitel
  • E-shop ani objednávkový systém
  • Blog ani sekce s články
  • Vícejjazyčná mutace
  • Ongoing údržba a aktualizace (lze doobjednat separátně)
  • Tvorba obsahu (texty a fotografie dodává objednatel)
  • PPC kampaně a správa reklam
  • Tvorba loga (lze doobjednat za příplatek)
