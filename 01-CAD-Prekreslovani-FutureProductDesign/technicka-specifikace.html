1. Uvod a cil projektu

Predmetem tohoto dokumentu je technicka specifikace reseni pro automatizovanou digitalizaci rucne kreslenych technickych vykresu do formatu CAD (DWG/DXF). Reseni vyuziva moderni metody pocitacoveho videni a umele inteligence k vyraznemu zrychleni a zlevneni procesu prekreslovani technickych vykresu.

Cilem je vytvorit webovy nastroj, ktery umozni nahrat sken nebo fotografii rucniho vykresu a automaticky vygenerovat vektorovy vystup ve formatu DXF/DWG, ktery nasledne konstrukter pouze zkontroluje a dopravi.

3. Faze projektu

Faze 1 - MVP

AI konverzni pipeline + webove rozhrani

Termin: 5 tydnu od zahajeni

  • Upload modul: Drag & drop nahravani souboru (JPG, PNG, PDF) s automatickou detekci kvality skenu
  • Preprocesing: Automaticka korekce perspektivy, kontrastu, odstraneni sumu a pozadi
  • AI rozpoznani: Detekce primych car, oblouku, kruhu, krizeni a prusecikovych bodu pomoci konvolucnich neuronovych siti
  • OCR modul: Rozpoznani kotovacich cisel, popisku a technickych znacek
  • DXF export: Generovani validniho DXF souboru s vrstvami (obrysy, koty, popisky, osy)
  • Webove rozhrani: Prehledny dashboard s historii konverzi, nahledem pred/po a moznosti stazeni vystupu
Faze 2 - Rozsireni

Manualni korekce a pokrocile funkce

Termin: Prubezne, dle dohody

  • Webovy editor: Integrovany vektorovy editor pro rucni upravu rozpoznanych entit primo v prohlizeci
  • Anotacni nastroje: Moznost manualne doplnit chybejici koty, znacky a popisky
  • Zpetna vazba AI: Korekce konstruktera se pouziji pro zlepsovani AI modelu
  • DWG export: Plna podpora DWG formatu vedle DXF
  • Davkove zpracovani: Hromadny upload a konverze vice vykresu najednou
Faze 3 - Optimalizace

Databaze sablon a automatizace

Termin: Dle dohody a rozsahu spoluprace

  • Knihovna sablon: Databaze opakujicich se prvku (srouby, matice, profily, standardni dily)
  • Automaticke rozpoznavani: AI identifikuje standardni dily a vlozi je jako bloky/referencni elementy
  • Podpora norem: Automaticke pridavani ramecku, razitka a popisovych poli dle CSN/ISO norem
  • API integrace: REST API pro napojeni na externi systemy (ERP, PLM)

4. Technicke prostredi

Komponenta Technologie Poznamka
Frontend Webova aplikace (SPA) Responsive, funguje na PC i tabletu
Backend Python / Django REST Framework API pro zpracovani a spravu konverzi
AI Engine PyTorch / OpenCV / Tesseract OCR Detekce car, OCR, preprocesing
CAD Export ezdxf (Python knihovna) Generovani validnich DXF souboru
Databaze PostgreSQL Metadata konverzi, uzivatele, historie
Ukladani souboru Cloudflare R2 / S3 Vstupni obrazky a vystupni DXF/DWG
Hosting Cloudflare Workers / VPS Dle ocekavane zateze

5. Podporovane formaty

Smer Format Poznamka
Vstup JPG, PNG Fotografie a skeny, min. 300 DPI doporuceno
Vstup PDF Vicestrankove dokumenty, automaticke rozdeleni na stranky
Vystup (Faze 1) DXF AutoCAD DXF R2018, kompatibilni se vsemi CAD systemy
Vystup (Faze 2+) DWG Nativni AutoCAD format
Nahled SVG, PNG Vizualizace v prohlizeci, overlay pres original
