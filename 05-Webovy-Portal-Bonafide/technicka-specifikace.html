1. Přehled projektu

Předmětem je vývoj obsahového webového portálu zaměřeného na bydlení, reality a architekturu. Portál bude sloužit jako moderní redakční platforma s integrací umělé inteligence (Google Gemini) pro asistenci při tvorbě a správě obsahu.

Cílem MVP je funkční portál s redakčním systémem, AI asistentem pro redaktory, veřejným frontendem pro čtenáře, uživatelskými účty a administračním panelem.

3. Moduly MVP

3.1 Redakční systém Backend

  • WYSIWYG editor s podporou obrázků, videí a formátovaného textu
  • Workflow správy článků: Draft → Review → Published
  • Kategorie: Bydlení, Reality, Architektura, Interiéry (rozšiřitelné)
  • Tagovací systém s autocomplete
  • Verzování obsahu (historie změn)
  • Plánované publikování (scheduled publish)
  • Náhledové URL pro nepublikované články

3.2 AI asistent pro redaktory Gemini

  • Automatický návrh titulků (3-5 variant) na základě obsahu článku
  • Sumarizace dlouhých článků (perex / abstrakt)
  • SEO optimalizace textu — návrh meta title, meta description, klíčových slov
  • Kontrola čitelnosti a doporučení pro zlepšení
  • Doporučování souvisejících článků na základě obsahu
  • AI-generovaný obsah vždy označen a vyžaduje schválení redaktorem

3.3 Veřejný frontend Angular

  • Homepage s featured články a kategoriemi
  • Listing článků s paginací a filtrováním
  • Detail článku s AI-generovaným shrnutím
  • Full-text vyhledávání s filtrováním podle kategorií
  • Responsivní design (mobile-first)
  • Lazy loading obrázků, skeleton loading
  • Open Graph meta tagy pro sdílení na sociálních sítích

3.4 Uživatelské účty

  • Registrace (e-mail + heslo), potvrzení e-mailem
  • Přihlášení (JWT tokeny)
  • Zapomenuté heslo (reset e-mailem)
  • Uživatelský profil — oblíbené články, historie čtení
  • Role: Čtenář, Redaktor, Šéfredaktor, Administrátor

3.5 Administrační panel

  • Správa uživatelů a rolí
  • Statistiky čtenosti (počet zobrazení, unikátní návštěvníci)
  • Správa kategorií a tagů
  • Přehled AI využití (počet volání, tokeny)
  • Dashboard s klíčovými metrikami

5. Bezpečnost Kritické

  • HTTPS — veškerá komunikace šifrována (TLS 1.3)
  • CSRF ochrana — Django middleware, SameSite cookies
  • Rate limiting — ochrana API před zneužitím (throttling per user/IP)
  • Sanitizace vstupů — ochrana proti XSS, SQL injection
  • JWT autentizace — access + refresh tokeny, secure httpOnly cookies
  • CORS — whitelisting povolených domén
  • Content Security Policy (CSP) — ochrana proti injektování skriptů
  • Logování — auditní log přístupů a změn obsahu
  • GDPR — souhlas s cookies, právo na smazání účtu, export dat

6.1 Optimalizační strategie

  • CDN (Cloudflare) pro statické soubory — obrázky, CSS, JS
  • Lazy loading obrázků s placeholder barvou
  • Server-Side Rendering (Angular Universal) pro SEO
  • Database indexy na frequently queried fields
  • Redis cache pro API responses (článkový listing, kategorie)
  • Image optimization — WebP formát, responsive sizes
  • Gzip/Brotli komprese

7. Hosting a infrastruktura

  • Produkce: Cloudflare Workers pro edge computing
  • Databáze: Managed PostgreSQL (Neon / Supabase alternative)
  • Cache: Redis (Upstash nebo obdobný managed service)
  • Soubory: Cloudflare R2 pro úložiště médií
  • CI/CD: GitHub Actions — automatický deploy na push do main
  • Monitoring: Cloudflare Analytics + Sentry pro error tracking
